
## 常用选项及其含义

| 常用选项    | 含义                     |
| ------- | ---------------------- |
| -E      | 只做预处理                  |
| -c      | 只编译不链接，生成目标文件 ". o"    |
| -S      | 生成汇编代码                 |
| -o file | 把输出生成到由 file 指定文件名的文件中 |
| -g      | 在输出的文件中加入支持调试的信息       |
| -v      | 显示输出详细的命令执行过程信息        |
