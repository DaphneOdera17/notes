

一个字符

```c
char c = 't';
```



一个字符串

```
char s[] = "test";
```

相当于

```
char s[] = {'t', 'e', 's', 't', '\0'};
```



$'\0'$ 为 $\NULL$ 终止符，表示字符串的结尾



$char*$ 存储字符串的起始位置

```c
char *p = &(s[0]);
```

也相当于

```c
char *p = s;
```



```c
*(p + 0) == 't';
*(p + 1) == 'e';
*(p + 2) == 's';
*(p + 3) == 't';
*(p + 4) == '\0'; \\ NULL
```

也相当于

```
p[0] == 't';
p[1] == 'e';
p[2] == 's';
p[3] == 't';
p[4] == '\0'; \\ NULL
```



引用字符串文字的内存位置

```c
const char *myString = "test";
```

之后，不能对字符串文字进行修改



通过循环遍历拷贝一个数组的时候应该从 $[0 \sim strlen(s)]$，结尾的 $’\0'$ 也应该被拷贝进去
